Why?

Because Flipboard is cool. It's easy to use. It gives an amazing oppurtunity for people who enjoy reading news daily. However.. their desktop app is kinda laggy and a bit too unaesthetic for my taste. That's why I wanted to create my own app, which might or might not be helpful for other people out there as well.

What did you use for this?

TL;DR I used Electron Framework for project. Of course, this wouldn't happen if Flipboard had a crappy web-page. Thank god, they have a really modern and fine design!

If you want more information, I can clear things up a little bit more of course. I used Electron's starter app as my scaffolding. I build up my own UI over it by discarding frame option. Most of the operations are working in Webview. For packaging, I used electron-builder, which is kinda complicated but extremely useful.

Any more features that you want to add?

I have a couple of ideas on my mind. But I don't want to involve in Flipboard's internal business. I already injected a CSS in the page to make it look more *unified*
